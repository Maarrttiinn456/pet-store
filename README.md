# Pet Store

React + Vite aplikace pro práci s Petstore API. Umožňuje přihlášení/registraci, filtrovaný seznam zvířat a simulované přidání zvířete.

## Live demo

Projekt běží na Netlify: https://pet-store-cz.netlify.app/

## Funkce

- Přihlášení a registrace (formy s validací)
- Seznam zvířat s filtrováním podle stavu
- Simulované přidání zvířete (API Petstore reálně neukládá nové záznamy)
- React Query pro práci s API

## Spuštění lokálně

1. Instalace závislostí:
   `npm install`
2. Spuštění dev serveru:
   `npm run dev`
3. Aplikace poběží na:
   `http://localhost:5173/`

## Technologie

- React + Vite
- TypeScript
- React Query
- React Hook Form + Zod
- Radix UI / shadcn-ui komponenty
